Supermarket Sales Form After reading barcode, cursor focusing column in grid is now on quantity based on the value mentioned on Cursor Focusing Order set in Application Parameters. The available values are 0,1,2 and 3. 0 is the default. If cursor focusing set as 0, focus will shift to the next row after reading barcode. If 1, focus will be on quantity column in retail only. If 2, focus will be on quantity column in wholesale only. if 3, focus will in quantity column in wholesale or retail. Feature
