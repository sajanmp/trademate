Trademate is a highly powerful, flexible and easy to use billing, inventory cum accounting software. It is useful and ideal for supermarkets, distributors, wholesalers, retailers, textile shops etc. The software is GST enabled. The interfaces are simple and easy to understand and use. The software comes with complete Accounts Module.

Trademate is a Windows based desktop application software developed by Icon Infoways. The company is based in Angamaly, Ernakulam District and founded in 2002.

Trademate uses the powerful Microsoft Sql Server Database engine for storing data. Trademate can connect to Sql Server databases hosted in cloud as well. Frequent updates are provided in Trademate as and when required based on customer feedbacks and industry needs. At present over 100 clients are using the software in the country
