Features

  • Interfaces are simple and easy to use and follow common pattern
  • Reports are highly flexible with sorting, searching etc
  • Drilldown provided in reports to trace the source document
  • Report columns can be resized, hidden and status saved
  • Reports support fast printing
  • Reports can be saved in excel or pdf format
  • Context sensitive lookups with sort/search
  • Automatic data backups
  • Easy integration with mobile app
  • Easy import of master data from excel sheets
  • Users and Access Levels
